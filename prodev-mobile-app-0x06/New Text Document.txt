2. Styling with NativeWind (TailwindCSS)
mandatory
Objectives

Tailwind CSS helps speed up development with ready-to-use utility classes. However, React Native doesn’t support Tailwind by default. To use it, we need NativeWind, a tool that makes Tailwind work smoothly in React Native. Setting it up isn’t straightforward, so it’s important to follow the installation guide carefully here (https://www.nativewind.dev/getting-started/expo-router).



For this task, we’ll continue working on prodev-mobile-app-0x05 and style it using Tailwind CSS.
